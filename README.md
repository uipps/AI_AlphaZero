# 这是中国象棋，python2版本
  暂不支持python3，本程序在python2.7下运行通过。 具备基本棋盘和走子功能，单机版人人对战，非人工智能版。
  这是的《人工智能导论》的最后课程项目，参考代码github上Linzertorte/ChineseChess，在此感谢！

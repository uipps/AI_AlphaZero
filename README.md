# AI_AlphaZero
人工智能，深度学习，基于AlphaZero的算法，实现五子棋、中国象棋、围棋等的自学习，还将尝试用AlphaZero实现青蛙过河、野人过河、走迷宫、两人取火柴游戏等......
不同分支代表不同游戏，例如gomoku、gobang就是五子棋；chess_chinese ( Chinese chess )就是中国象棋；rubikscube就是魔方，又分二阶、三阶等
